# Student Management REST API

A backend REST API built using **Python, Flask, and SQLite**.

## Features
- Create student
- View all students
- View single student
- Update student marks
- Delete student

## Tech Stack
- Python
- Flask
- SQLite

## API Endpoints
- GET /students
- GET /students/<id>
- POST /add-student
- PUT /update-student/<id>
- DELETE /delete-student/<id>
